系统简介

本系统是一个基于 Java Swing 的学生信息管理应用，严格遵循面向对象设计原则， 采用 继承 + 组合 架构。系统支持本科生与研究生管理、学科动态绑定、 成绩录入与自动计算、多维度查询、班级排名及统计分析。 界面风格采用校园科技风配色方案——校徽深蓝侧边栏导航 + 白色卡片式布局。

面向对象约束

  • 继承：Student → Undergraduate / Graduate
  • 组合：Student 包含 Address
  • 静态变量：totalCount / undergradCount / gradCount
  • 多文件分包：entity / service / ui
  • 边界处理：判满 (200上限) / 判空 / 非法输入

技术栈

  • 语言：Java (JDK 21)
  • UI 框架：javax.swing (原生 Swing)
  • 架构：entity → service → ui 三层
  • 数据存储：内存 Map
  • 构建：IntelliJ IDEA

特 核心特性与创新点

继承 + 组合

Student 抽象父类定义公共属性和方法，Undergraduate / Graduate 分别继承并扩展。Address 以组合方式嵌入 Student。

学科动态绑定

按「专业 + 年级」统一管理学科列表。录入成绩时自动加载对应学科，同专业同年级共享同一学科组。

班级排名

支持总分排名和单科排名，前三名高亮显示（金/银/铜配色）。排名范围限定为同年级+同专业+同班级。

边界防御

学号唯一性校验、系统容量上限 (200)、判空判满、非法输入自动拦截，所有错误以红色提示。

校园科技风 UI

Swing 原生组件深度定制——圆角渐变按钮、阴影卡片、校徽蓝侧边栏、前三名特殊着色、表格悬停效果。

静态变量统计

totalCount / undergradCount / gradCount 实时追踪人数变化，增删学生自动更新，统计面板即时刷新。

public abstract class Student {
  protected String studentId, name, grade, clazz, major;
  protected int age;
  protected Address address;  // 组合
  protected Map<String, Double> scores;

  // 静态变量 — 人数统计
  public static int totalCount = 0;
  public static int undergradCount = 0;
  public static final int MAX = 200;

  // 抽象方法 — 子类实现
  public abstract String getTypeLabel();
  public abstract String getInfo();

  // 计算总分 / 平均分
  public double calculateTotal() { ... }
  public double calculateAverage() { ... }
}

  private String researchDirection;  // 新增：研究方向

    super(studentId, name, age, grade, clazz, major, address);

    this.researchDirection = researchDirection;

  }

  @Override
  public String getTypeLabel() { return "研究生"; }

  @Override
  public String getInfo() {
    // 格式化输出含导师、研究方向的信息卡片
    ...
  }
}
